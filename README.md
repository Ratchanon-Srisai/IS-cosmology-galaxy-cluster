# is-cosmology-galaxy-cluster
A forward-modeling galaxy cluster cosmology pipeline using PyCCL for cosmological parameter constraints, developed as an Undergraduate Research Project. การวิเคราะห์เอกภพวิทยาด้วยกระจุกกาแล็กซีแบบการจำลองไปข้างหน้า โดยใช้ PyCCL ในการจำกัดค่าพารามิเตอร์ทางเอกภพวิทยา (เป็นส่วนหนึ่งของรายวิชาโครงงานวิจัยปริญญาบัณฑิต)
